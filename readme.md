* [Redux](https://redux.js.org/)


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ClawGibson/FrontEnd-Test
   ```
2. Install dependencies
   ```sh
   yarn
   ```
3. Run
   ```.env
   yarn start
   ```
